How We Match

We run two passes with a ±30-day window and deposit/withdrawal direction matching:

— Exact: Same check number + same amount + same account + within 30 days.

— Close: Same amount + within 30 days + same account, even if the check number doesn't line up.

— Direction: Deposits only match deposits, withdrawals only match withdrawals. No cross-matching.

Everything else gets flagged as a gap. BMO-TTICorp and BMO-TTIPark both map to the same GL account, so they match as a combined pool.

The Big Picture
